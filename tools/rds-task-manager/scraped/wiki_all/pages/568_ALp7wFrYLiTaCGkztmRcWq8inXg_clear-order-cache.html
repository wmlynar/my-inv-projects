Clear Order Cache​

Modified May 17, 2024
Request​
•
Function: Clear the database waybill cache, which can clear the waybill cache before the specified timestamp.​
•
Method: post​
•
Interface Description: /clearCache​
​
Code block​
POST "http://host:8088/clearCache"​
​
Request data​
​
Name​
Type​
Description​
Required​
type​
string​
The cache types that need to be cleared can currently be ordinary waybills = "orders" and composite waybills = "simpleOrders".​
Yes​
timestamp​
int​
The format is Unix timestamp, with the unit being seconds (s). It represents the waybill that needs to be cleared before the specified timestamp and is in the final state.​
Yes​
​
There are two ways to clear the waybill cache. You can choose one of them when sending:​
1.
Clear normal waybill​Create Order cache.​
2.
Clear flattened single waybill ​Create Join Ordercache.​
You can use the timestamp conversion tool for timestamp conversion.​
