绕行（沿线路）​

Created on March 24, 2024
950
1676
1

Feishu user 3564
​
场景定义​
机器人在运行时遇到障碍物或故障机器人时，调度会为机器人重新规划出一条新的路线来绕过障碍物或故障机器人​
绕行逻辑​
沿线路的绕行路线严格遵循线路的拓扑结构及路线方向，现有绕行可解决以下情形：​
1.
遇到障碍物时的绕行​
2.
被故障机器人阻挡时的绕行（故障机器人：急停、rbk报错、不明原因的原地不动等）​
3.
调度报错 x is blocked by y 一段时间后，x 会绕路 ​
4.
绕障碍物的过程中遇到障碍物​
5.
绕障碍物的过程中遇到故障机器人​
6.
绕故障机器人的过程中遇到障碍物​
7.
绕故障机器人的过程中遇到故障机器人​
