查询外部互斥组请求历史​

Modified December 20, 2024
请求：​
•
功能：查询指定机器人或互斥组，在一段时间内，对外部互斥组的请求历史​
•
方法： GET​
•
路径： /trace/otherSysBlockGroup​
​
Code block​
GET "http://host:8088/trace/otherSysBlockGroup"​
​
参数介绍​
​

Type​
Description​
Required​

int​

否​

int​

否​
orderBy​
string​
排序列名，支持​
否​
orderMethod​
string​
排序方式，支持 descending / ascending​
否​
where​
string​
过滤条件​
否​
​
支持排序和过滤的列：​
•
element_id​
•
event_type​
•
timestamp​
•
msg​

​
