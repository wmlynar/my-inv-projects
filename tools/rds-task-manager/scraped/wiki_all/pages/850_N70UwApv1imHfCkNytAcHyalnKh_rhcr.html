RHCR多车路径规划算法​

Modified July 14
​
Feishu user 7037
​
除G-MAPF外，Core提供另一种多车路径规划算法：RHCR​
使用方法​
1.
需要使用版本号前缀为0.2.0的Core！​
2.
在参数配置中，先将G-MAPF改为False；此处需要注意，机器人执行任务时，G-MAPF无法关闭，需要等到任务执行完毕，G-MAPF才会“释放”机器人，所有机器人都释放后，报警码54006消失，G-MAPF才算成功关闭​
3.
在参数配置中， 将 RHCR 改为True 240902后默认开启​
4.
参数相关问题，需要咨询调度团队​
