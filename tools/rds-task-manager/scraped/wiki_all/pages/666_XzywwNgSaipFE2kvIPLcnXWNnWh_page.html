创建运单​

Modified July 22
机器接口介绍​
此 API 将创建一个运单，并向 RDSCore 提交一个 “对机器人的申请”（注意，运单并非能立即获取到机器人）。​
此接口可以发送两种不同形式的运单， 普通运单 和 拼合单 。此处为发送 普通运单 的详情。 普通运单 和 拼合单 发送的数据结构不同，不可混合发送。​
​
此接口可以发送两种不同形式的 普通运单 ，目的地为 地图点位 的运单和目的地为 地图库位 的运单。此处介绍下图中 红色方框 框住的部分。​
​
​

•
