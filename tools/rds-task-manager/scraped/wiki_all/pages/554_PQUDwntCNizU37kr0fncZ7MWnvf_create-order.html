Create Order​

Modified May 17, 2024
Introduction​
This API will create an order and submit it as an "Request for Robot" to RDSCore (note that the order does not immediately assignable to a robot).​
This interface can send two different forms of order, normal order and ​Create Join Order. Here are the details for sending a normal order . The data structures sent by the normal order and the​Create Join Order are different, and they cannot be sent mixed.​
​
This interface can send two different forms of normal order , the order with the destination of the map point location and the order with the destination of the map storage bin location . The part enclosed by the red box in the image below is expand described here.​
​
