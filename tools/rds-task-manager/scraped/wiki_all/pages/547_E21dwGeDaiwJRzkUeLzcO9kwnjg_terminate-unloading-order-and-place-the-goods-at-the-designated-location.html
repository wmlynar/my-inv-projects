Terminate Unloading Order and Place the Goods at the Designated Location​

Modified April 28, 2024
1.
Terminate the order and set the robot to not accept orders.​
​
​
2.
Check if the robot is carrying goods.​
​Inquire Robot Status The field isLoaded indicates whether to carry goods.​
3.
If there is no cargo, set the robot to accept orders and end.​
4.
If carrying goods, reallocate all orders pre-allocated to robots. ​Redistribute Order​
5.
Send an unloading order.​
You need to specify the robot and raise the priority of the order. Example:​
​
Code block​
{​
"blocks": [​
{​
"blockId": "{$2$}-c",​
"location": "AP12",​
"operation": "JackUnload"​
}​
],​
"complete": true,​
