机器人定位状态​
uint16​
00008​
——​
0 = 定位失败,​
1 = 定位正确,​
2 = 正在重定位,​
3 = 定位完成；​
机器人刚启动后，定位状态为 2 ，当定位状态变为 3 时，可以用 [0x]00003 地址位确认定位正确，确认后定位状态将变为 1​

1 = 等待执行导航,​
2 = 正在执行导航,​
3 = 导航暂停,​
4 = 到达,​
5 = 失败,​
6 = 取消,​
7 = 超时​
机器人当前导航类型​
uint16​

——​
0 = 没有导航,​
1 = 自由导航到任意点,​
2 = 自由导航到站点,​
3 = 路径导航到站点,​
7 = 平动转动,​
100 = 其他​
