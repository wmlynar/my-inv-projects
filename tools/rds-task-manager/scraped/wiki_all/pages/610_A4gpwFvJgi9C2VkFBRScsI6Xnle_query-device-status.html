Query Device Status​

Modified May 16, 2024
Request​
•
Function: Query device status.​
•
Method: Get​
•
Interface Description: /devicesDetails​
​
Code block​
GET "http://host:8088/devicesDetails"​
​
Request parameters​
​
Name​
Type​
Description​
Required​
devices​
string​
The types of devices that need to be queried are separated by a comma. Currently, they can only be doors, lifts, terminals, and windshowers. If no query parameters are filled in, all devices will be queried by default.​
No​
​
Request example​
Check doors and elevators​
​
Code block​
GET "http://host:8088/devicesDetails?devices=doors,lifts"​
​
Query all devices​
​
Code block​
GET "http://host:8088/devicesDetails"​
​
Response​
Response data​
​
Name​
Type​
Description​
doors​
object array​
Object array of door information, each object represents a door, see doors.​
terminals​
object array​
Object array of terminal information, each object represents a terminal, see terminals.​
