部署在控制器（SRC-2000）里面​

Modified June 21, 2024
此版本会基于 Docker 将 RDS 和 Core 安装到 SRC-2000 内，适用于通过叉车 APP 直接控制机器人而无需把 RDS 和 Core 安装到其他服务器。​
必要条件​
SRC 需要安装的工具/软件：​
1.
rinetd：用于重定向 TCP 端口流量的工具​
2.
Docker：用来解决 Core 和 SRC 程序端口冲突问题​
3.
RDS + Core 的 Docker 镜像​
4.
SRC 远程连接的 SSH 密钥（向 @苏国庆 获取）​
​
远程登录控制器​
使用任何一个远程工具，配置获取到的 SSH 密钥进入 SRC 控制器的命令行界面。​
安装 rinetd​
管理 Robokit 的端口映射，确保在机器人入网之后，其他网络设备，可以通过无线网络访问机器人。​
在线安装​
在控制器终端执行：​
​
Code block​
sudo apt-get update​
sudo apt-get install rinetd​
echo "0.0.0.0 8080 192.168.193.15 8080" | sudo tee -a /etc/rinetd.conf​
echo "0.0.0.0 8088 192.168.193.15 8088" | sudo tee -a /etc/rinetd.conf​
echo "0.0.0.0 7600 192.168.193.15 7600" | sudo tee -a /etc/rinetd.conf​
sudo systemctl restart rinetd​
​
离线安装​
1.
下载离线安装包 📎rinetd_0.62.1sam-1_amd64.zip ，​
2.
解压压缩包，并将解压后得到的安装包 rinetd_0.62.1sam-1_amd64.deb 上传到控制器。​
3.
在安装包所在路径下，执行如下指令：​
​
Code block​
sudo dpkg -i rinetd_0.62.1sam-1_amd64.deb​
​
echo "0.0.0.0 8080 192.168.193.15 8080" | sudo tee -a /etc/rinetd.conf​
echo "0.0.0.0 8088 192.168.193.15 8088" | sudo tee -a /etc/rinetd.conf​
echo "0.0.0.0 7600 192.168.193.15 7600" | sudo tee -a /etc/rinetd.conf​
sudo systemctl restart rinetd​
​
