设置继电器​

Modified May 3, 2024
​
​
🎨
注意：​
该 API 用于设置机器人控制器上的继电器状态。（目前只有SRC2000控制器有该接口）​
​
​
请求​
​
•
编号: 6003 (0x1773)​
•
名称: robot_other_setrelay_req​
•
描述: 设置继电器​
•
JSON 数据区: 见下表​
​
字段名​
类型​
描述​
可缺省​
status​
boolean​
true 为继电器吸合, false 为继电器断开​
否​
​
​
请求示例​
​
​
Code block​
5A 01 00 01 00 00 00 0F 17 73 00 00 00 00 00 00​
{"status":true}​
// 设置继电器为吸合​
// 序列化后十六进制为：​
7B 22 73 74 61 74 75 73 22 3A 74 72 75 65 7D​
​
​
响应​
​
•
编号: 16003 (0x3E83)​
•
名称: robot_other_setrelay_res​
•
描述: 设置继电器的响应​
•
JSON 数据区: 见下表​
​
字段名​
类型​
描述​
可缺省​
