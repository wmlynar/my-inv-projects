配置机器人推送端口​

Modified May 3, 2024
​
请求​
​
•
编号: 9300​
•
名称: robot_push_config_req​
•
描述: 配置推送端口 19301​
•
JSON 数据区: 见下表​
​
字段名​
类型​
描述​
可缺省​
interval​
Interger​
消息推送时间间隔（ms）​
是​
included_fields​
array[string]​
设置消息中包含的字段​
是​
excluded_fields​
array[string]​
设置消息中排除的字段​
是​
​
​
​
🥖
警告：​
included_fields 和 excluded_fields 不可同时设置​
​
​
请求示例​
​
​
Code block​
// 1）设置发布时间间隔为 2000 ms​
{"interval":2000}​
​
// 2）设置 include_fields​
{"included_fields":["vx","vy"]}​
​
// 3) 设置 excluded_fields​
{"excluded_fields":["blocked"]}​
​
// 4) 同时设置发布时间间隔 和 include_fields​
{"interval":2000,"included_fields":["vx","vy"]}​
​
