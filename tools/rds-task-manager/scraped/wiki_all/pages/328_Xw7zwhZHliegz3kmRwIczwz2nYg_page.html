Modified August 28, 2024
（3.4.6.13+）​
请求​
​
•
编号: 3059 (0x0BF3)​
•
名称: robot_task_path_req​
•
描述: 启用和禁用线路​
•
JSON 数据区: 见下表​
a.
禁用线路​
​
字段名​
类型​
描述​
可缺省​
disablePath​
json string​
禁用的线路名称，json string 格式为{"id":"线路名称"}​
否​
​

字段名​
类型​
描述​
可缺省​
enablePath​
json string​
启用的线路名称，json string 格式为{"id":"线路名称"}​
否​
​
c.
查询线路​
见请求示例。​
