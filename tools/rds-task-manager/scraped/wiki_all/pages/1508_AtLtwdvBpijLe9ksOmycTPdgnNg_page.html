机器人故障记录​

Modified August 1, 2024
接口介绍​
此 API 可以查询机器人故障记录。​
请求​
•
功能：查询机器人故障记录​
•
方法：POST​
•
接口说明： /api/stat/alarmsRecords​
•
请求示例的 URL​
​
Code block​
POST http://host:8080/api/stat/alarmsRecords​
​
请求参数​
​
Name​
Type​
Description​
Required​
currentPage​
int​
表示查询第 currentPage 页的结果​
是​
pageSize​
int​
表示按照每页 pageSize 条记录，对所有结果进行分页​
是​
queryParam​
Object​
查询参数对象​
是​
startCreateTime​
String​
故障开始时间的开始（与endCreateTime，必须两个都填或两个都不填）​
否​
endCreateTime​
String​
故障开始时间的结束（与startCreateTime，必须两个都填或两个都不填）​
否​
startEndTime​
String​
故障结束时间的开始（与endEndTime，必须两个都填或两个都不填）​
否​
endEndTime​
String​
故障结束时间的结束（与startEndTime，必须两个都填或两个都不填）​
否​
vehicleId​
String​
机器人Id​
否​
level​
String​
故障等级(errors:错误；warnings:告警)​
否​
