批量放行​

Modified March 24, 2024
接口介绍​
通过调用此接口，根据agvId，放行当前正在等待放行的agv​
请求​
•
功能：当天风任务使用了等待放行块之后，根据agvId，放行当前正在等待放行的agv​
•
方法： POST​
•
接口说明： /api/releaseWaitPassBlock​
•
请求示例URL POST http://localhost:8080/api/releaseWaitPassBlock​
请求参数 ​
​
*Name**​
Type​
Description​
Required​
agvIds​
List​
需要放行的agvid的列表​
是​
​
请求示例​
放行当前正在等待放行的agv​
​
Code block​
POST http://localhost:8080/api/releaseWaitPassBlock​
​
["sim_01","sim_02"]​
​
