查询机器人完成的任务列表​

Modified March 24, 2024
适用 rds-F1.9.5-B1.7.20 及以上​
查询机器人完成的任务列表​
接口介绍​
此 API 可以查询机器人完成的任务列表。​
请求​
•
功能：查询机器人完成的任务列表​
•
方法：POST​
•
接口说明： /api/stat/agvTaskSuccessList​
•
请求示例的 URL​
​
Code block​
POST http://host:8080/api/stat/agvTaskSuccessList​
​
请求参数​
​
Name​
Type​
Description​
Required​
start​
String​
统计的开始时间， yyyy-MM-dd 格式。​
是​
end​
String​
统计的结束时间， yyyy-MM-dd 格式。​
是​
​
请求示例​
查询 2020-01-01 00 : 00 : 00 - 2022-07-19 23 : 59 : 59 完成的任务列表​
​
Code block​
{​
"start":"2020-01-01",​
"end":"2022-07-19"​
}​
​
