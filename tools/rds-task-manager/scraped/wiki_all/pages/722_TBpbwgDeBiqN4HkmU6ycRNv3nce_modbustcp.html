代理ModbusTCP请求​

Modified March 24, 2024
请求​
•
功能：使用Core的连接来代理ModbusTCP请求，避免过多连接导致的问题​
•
方法： POST​
•
接口说明： /modbusProxyRequest​
​
Code block​
POST "http://host:8088/modbusProxyRequest"​
​
请求数据​
​
Name​
Type​
Description​
Required​
proxyName​
string​
代理名称，对应模型文件-proxy中配置的设备名称​
是​
address​
int​
地址位, 可以为0-65535​
是​
functionCode​
int​
ModbusTCP函数号,可以为1,2,3,4,5,6,15,16​
是​
number​
int​
写入/读取的数量​
否​
value​
int[]​
写入的值​
否​
​
请求数据示例​
​
Code block​
{​
"proxyName":"D1",​
"address":101,​
"functionCode":1,​
"number":10​
}​
​
