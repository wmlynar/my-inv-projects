Unbind Goods from Specified Containers​

Modified May 16, 2024
Interface introduction​
The API is consistent with the robot API ​Unbind Goods from Specified Containers function.​
Request​
•
Function: Unbind the goods from the specified container. This interface is only effective for robots with containers configured in the robot model file. The relevant information of the container can be obtained in the containers field of the robot information.​
•
Method: post​
•
Interface Description: /clearContainer​
​
Code block​
POST "http://host:8088/clearContainer"​
