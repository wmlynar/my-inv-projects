银河麒麟​
​
安装方式:​
1.
把 Roboshop.deb 拷贝到桌面 （ 注: Roboshop.deb 仅代指安装包 ）​
2.
安装方式​
a.
鼠标双击 Roboshop.deb 安装​
b.
命令行安装:​
i.
鼠标在桌面空白处右击显示菜单，选中【打开终端】或【Open in Terminal】​
ii.
安装命令 sudo dpkg -i ./Desktop/Roboshop.deb （ 注: Roboshop.deb 要改成实际文件名 ） ​
安装完成后不要使用 sudo 启动 Roboshop, 否则文件会带锁，鼠标双击图标启动会出现文件没有写入权限问题.​
去锁命令: sudo chmod -R 777 /opt/Roboshop/​
卸载命令: sudo dpkg -r roboshop​
​
3.
点击 Roboshop 图标运行 ​
