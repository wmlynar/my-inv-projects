查询设备状态​

Modified March 24, 2024
接口介绍​
此 API 通常用于 AGV 与客户设备通过 RDSCore 交互。此时在 Roboshop Pro 配置界面中需要将终端类型配置为 slave 。在此模式下 RDSCore 作为终端和modbus slave，同时接收小车与客户设备的指令。​
请求​
•
功能：查询设备状态​
•
方法： POST​
•
接口说明： /getTerminalStatus​
​
