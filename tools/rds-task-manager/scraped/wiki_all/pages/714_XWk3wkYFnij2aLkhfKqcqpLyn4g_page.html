机器人重定位​

Modified July 24
请求​
•
功能：重定位机器人​
•
方法： POST​
•
接口说明： /reloc​
​
Code block​
POST "http://host:8088/reloc"​
​
请求数据​
​
Name​
Type​
Description​
Required​
vehicle​
string​
需要重定位的机器人名​
是​
x​
double​
世界坐标系中的 x 坐标, 单位 m​
是​
y​
double​
世界坐标系中的 y 坐标, 单位 m​
是​
angle​
double​
世界坐标系中的角度, 单位 rad​
是​
length​
double​
重定位 区域半径，单位 m​
是​
​
请求示例​
​
Code block​
{​
"vehicle": "AMB-01",​
"x": 4.2,​
"y": 3.4,​
"length": 1.8,​
"angle": 1.7​
}​
​
