API 错误码​

Modified May 3, 2024
API 错误码为响应数据区 JSON 对象包含的错误码（ ret_code ），用于指示对请求执行不成功或请求出错等错误。若 ret_code 为 0 或缺省，说明没有错误，若不为0，说明发生了错误，错误码见下表：​
错误码只会出现在响应数据区的 JSON 数据中​
​

req_unavailable​

param_missing​

请求参数类型错误​
40003​

请求参数不合法​
40004​
mode_error​
运行模式错误​
