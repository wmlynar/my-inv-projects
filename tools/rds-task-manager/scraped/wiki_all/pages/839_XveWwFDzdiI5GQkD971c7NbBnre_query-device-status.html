Query Device Status​

Modified May 16, 2024
Interface introduction​
This API is usually used for AGVs to interact with client devices through RDSCore . At this time, in the configuration interface of Roboshop Pro , the terminal type needs to be configured as slave . In this mode, RDSCore acts as the terminal and modbus slave, and receives instructions from both the robot and the client device.​
