修改运单 label​

Modified March 24, 2024
请求​
•
功能：当前运单没有分派机器人时，可以下发修改运单的 label ，从而调整选车范围。 unassign 选项为true时，修改已分配未执行订单的标签，并且重分配订单。​
•
方法： POST​
•
接口说明： /setLabel​
​
Code block​
POST "http://host:8088/setLabel"​
​
请求数据​
​
Name​
Type​
Description​
Required​
id​
string​
修改 label 的运单 ID​
是​
label​
string​
修改后的 label​
是​
unassign​
bool​
重分配订单​
否​
​
请求示例​
​
Code block​
{​
"id": "task1", // 运单 task1​
"label": "fork",​
"unassign": true​
}​
​
响应​
响应数据​
​
Name​
Type​
Description​
code​
int​
API 错误码，详情见 API 错误码​
msg​
string​
API 错误码信息​
create_on​
string​
API 上传时间戳​
​
