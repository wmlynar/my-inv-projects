追加运单的动作块​

Modified December 26
接口介绍​
当 普通运单 创建、且未封口时，可以通过以下 API 来追加动作块。​
​
此接口可以向 普通运单 追加三种形式的 block，追加的 block 按顺序添加到 普通运单 数据中的 blocks 数组中，机器人会按照 blocks 数组顺序执行动作。 三种形式的 block 为下图中 红色方框 框住的部分。​
​
​
请求​
•
功能：追加运单的动作块​
•
方法： POST​
•
接口说明： /addBlocks​
​
Code block​
POST "http://host:8088/addBlocks"​
​
​
