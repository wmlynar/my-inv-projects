Modify Model and Paramerters​

Modified April 29, 2024
RDSCore and Robokit use the same set of TCP APIs, so you can query parameters through the status API and modify RDSCore's model file and parameter configuration through the configuration API. The configuration API uses TCP port 19207; the status API uses TCP port 19204.​
