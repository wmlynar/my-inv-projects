MAPF (Multi-Agent Path Finding), means that in the process of performing tasks, the AGV will not completely follow the principle of the shortest route, but will plan according to the current state of the surrounding AGVs, wait in place, detour and other actions to avoid deadlock and optimize the overall operation efficiency.â€‹

Definition of route intersection: Routes between two points intersect, and there are no shared connection points at the intersection.â€‹

When individual route intersections occur, using the method of drawing exclusion areas can ensure that MAPF does not result in deadlock at that location. However, if there are too many route intersections in a scene, MAPF is not suitable for use.â€‹
