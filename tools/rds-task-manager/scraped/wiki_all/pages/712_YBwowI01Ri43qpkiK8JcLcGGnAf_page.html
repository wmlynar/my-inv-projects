获取指定机器人的地图​

Modified March 24, 2024
请求​
•
功能：查询指定机器人的地图数据​
•
方法： POST​
•
接口说明： /robotSmap​
​
Code block​
POST "http://host:8088/robotSmap"​
​
请求数据​
​
Name​
Type​
Description​
Required​
vehicle​
string​
机器人名​
是​
map​
string​
地图名，注意需要带上后缀 .smap​
是​
​
请求示例​
​
Code block​
{​
"vehicle": "RD-TEST-1", // 机器人名​
"map": "default.smap" // 机器人地图​
}​
​
响应​
响应数据​
响应信息为机器人地图 json 数据，详情见 smap地图格式解析​
​
