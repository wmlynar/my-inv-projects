检查库位有效性​

Modified March 24, 2024
请求​
•
功能：查询场景中的库位的有效性​
•
方法： POST​
•
接口说明： /binCheck​
​
Code block​
POST "http://host:8088/binCheck"​
​
请求数据​
​
Name​
Type​
Description​
Required​
bins​
string array​
需要检查的库位名列表​
是​
​
当 bins 为空数组时，则默认检查场景中所有的库位，可以根据此功能获取所有库位绑定的点位以及库位配置的 binTask 动作。​
请求示例​
检查指定库位的有效性​
​
Code block​
{​
"bins": ["bin-02", "bin-04","bin-100"] // 库位名称​
}​
​
查询所有库位信息（绑定库位及 binTask 动作）​
​
Code block​
{​
"bins": []​
}​
​
​
响应​
响应数据​
​
Name​
Type​
Description​
bins​
object array​
库位状态列表，详情见 bins表​
