还原场景​

Modified March 24, 2024
请求​
•
功能：还原 RDSCore 的场景资源包为默认值​
•
方法： POST​
•
接口说明： /syncScene​
​
Code block​
POST "http://host:8088/syncScene"​
​
请求数据​
​
Name​
Type​
Description​
Required​
vehicles​
string array​
需要还原的机器人名数组，需RDSCore版本大于0.1.8.221123​
否​
​
默认不发任何数据为还原所有机器人。​
请求示例​
​
Code block​
{​
"vehicles": ["AMB-01", "AMB-02"] // 车名数组​
}​
​
响应​
响应数据​
​
Name​
Type​
Description​
code​
int​
API 错误码，详情见 API 错误码​
msg​
string​
API 错误码信息​
create_on​
string​
API 上传时间戳​
​

Responses Code 200​

​
